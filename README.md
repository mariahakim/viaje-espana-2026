# Viaje España 2026

App web móvil y privada para concentrar el itinerario, reservas, pendientes, notas y documentos del viaje.

## Abrir

Desde esta carpeta:

```bash
python3 -m http.server 4173
```

Luego abre `http://localhost:4173`.

La app guarda los pendientes y las notas únicamente en el navegador del dispositivo. Los datos sensibles pueden ocultarse con el botón circular de la esquina superior derecha.
